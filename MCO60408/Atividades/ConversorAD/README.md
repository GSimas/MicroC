# Conversor A/D

Circuito conversor analógico-digital, sensor de temperatura LM35, display LCD, microcontralador ATMEGA328P.
Simulação de Circuito no software [ISIS Proteus](https://www.labcenter.com/).
Código desenvolvido no software [ATMEL Studio](http://www.atmel.com/microsite/atmel-studio/)
